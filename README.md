# 感谢关注FxckVip

## 概述

FxckVip 是一个 XPOSED 模块，旨在帮助用户获取部分应用的 VIP 权限或去除广告。目前，该模块目前支持以下应用：

- **某茄小说**: 去除阅读广告
- **某猫小说**: 去除阅读广告，显示 VIP 状态

## 兼容性

- **已 Root 的设备**: 推荐使用 LSPosed 框架。
- **未 Root 的设备**: 可以尝试使用 LSPatch。
